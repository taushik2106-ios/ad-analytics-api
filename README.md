# Ad Analytics API (Python)

## Overview
This project is a Python-based REST API that simulates advertisement campaign data and provides analytics insights such as Click-Through Rate (CTR).

## Features
- Fetch advertisement campaign data
- Analyze ad performance metrics
- RESTful API endpoints using FastAPI

## Technologies Used
- Python
- FastAPI
- REST APIs
- JSON

## Endpoints
- `/` – API health check  
- `/ads` – Returns ad campaign data  
- `/analytics` – Returns CTR analysis for campaigns  

## Learning Outcome
- Built REST APIs in Python
- Worked with structured data
- Understood analytics concepts relevant to ad-tech systems
